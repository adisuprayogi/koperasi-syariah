## ğŸ¯ Cara Mengambil Screenshot

### Metode 1: Browser Developer Tools
1. Buka halaman yang diinginkan
2. Buka Developer Tools (F12)
3. Klik tombol screenshot di Device Toolbar
4. Pilih ukuran yang diinginkan
5. Save sebagai PNG

### Metode 2: Browser Extensions
Gunakan extensions seperti:
- **Awesome Screenshot** (Chrome)
- **Nimbus Screenshot** (Chrome/Firefox)
- **Lightshot** (Windows)

### Metode 3: System Screenshot
- **Windows**: Win + Shift + S
- **Mac**: Cmd + Shift + 4
- **Linux**: PrtScn atau aplikasi screenshot
