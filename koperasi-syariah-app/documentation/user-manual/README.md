## Pendahuluan

Aplikasi Koperasi Syariah adalah sistem informasi yang dirancang untuk mengelola kegiatan operasional koperasi syariah, meliputi pengelolaan anggota, simpanan, pembiayaan, dan pelaporan keuangan.

### Fitur Utama
- **Manajemen Anggota**: Registrasi dan data keanggotaan
- **Simpanan**: Simpanan pokok, wajib, dan sukarela
- **Pembiayaan**: Pengajuan dan monitoring pembiayaan syariah
- **Laporan**: Laporan keuangan dan transaksi
- **Mobile Friendly**: Akses mudah dari perangkat apapun

## Login dan Registrasi

### Registrasi Anggota Baru

1. Buka aplikasi di browser
2. Klik tombol "Register" di halaman login
3. Isi form registrasi:
   - **Nama Lengkap**: Nama lengkap sesuai KTP
   - **Email**: Email aktif untuk notifikasi
   - **No. Handphone**: Nomor WhatsApp aktif
   - **Tanggal Lahir**: Format DD-MM-YYYY
   - **Alamat**: Alamat lengkap
   - **Password**: Minimal 8 karakter
   - **Konfirmasi Password**: Ulangi password

4. Klik "Register" untuk menyelesaikan registrasi
5. Tunggu persetujuan dari admin koperasi

### Login ke Akun

1. Buka halaman login aplikasi
2. Masukkan email dan password
3. Klik "Login"
4. Jika berhasil, akan diarahkan ke dashboard

### Lupa Password

1. Klik "Lupa Password?" di halaman login
2. Masukkan email terdaftar
3. Check email untuk link reset password
4. Ikuti instruksi untuk reset password

## Dashboard Anggota

### Menu Navigasi
- **Beranda**: Dashboard utama

## Profil Pengguna

### Melihat Profil
1. Klik menu "Profil Saya"
2. Lihat informasi pribadi:
   - Nomor Anggota
   - Data identitas lengkap
   - Status keanggotaan
   - Foto profil

### Mengubah Profil
1. Klik tombol "Edit Profil"
2. Update informasi yang diperlukan:
   - No. Handphone
   - Alamat
   - Foto profil
3. Klik "Update Profil"

### Mengubah Password
1. Klik menu "Ubah Password"
2. Masukkan password lama
3. Masukkan password baru (minimal 8 karakter)
4. Konfirmasi password baru
5. Klik "Update Password"

## Data Simpanan

### Jenis-Jenis Simpanan

#### Simpanan Pokok
- **Deskripsi**: Simpanan wajib saat pertama kali mendaftar
- **Jumlah**: Tetap (ditentukan koperasi)
- **Bisa Ditarik**: Tidak, kecuali keluar dari koperasi

#### Simpanan Wajib
- **Deskripsi**: Simpanan bulanan wajib
- **Jumlah**: Tetap per bulan
- **Bisa Ditarik**: Tidak, kecuali keluar dari koperasi

#### Simpanan Sukarela
- **Deskripsi**: Simpanan tambahan sesuai kemampuan
- **Jumlah**: Bebas
- **Bisa Ditarik**: Ya, dengan syarat tertentu

### Melihat Data Simpanan
1. Klik menu "Data Simpanan"
2. Pilih tab sesuai jenis simpanan:
   - **Tab Rekap**: Total semua simpanan
   - **Tab Detail**: Transaksi per tanggal

### Informasi yang Ditampilkan
- **Saldo Saat Ini**: Total simpanan per jenis
- **Riwayat Transaksi**: Daftar setoran dan penarikan
- **Grafik Perkembangan**: Visualisasi pertumbuhan simpanan

### Download Bukti Setoran
1. Cari transaksi yang diinginkan
2. Klik ikon üñ®Ô∏è (print) di kolom aksi

## Pembiayaan

### Mengajukan Pembiayaan Baru

1. Klik menu "Pembiayaan"
2. Klik tombol "Ajukan Pembiayaan Baru"
3. Isi form pengajuan:
   - **Jenis Pembiayaan**: Pilih jenis sesuai kebutuhan
   - **Jumlah Pengajuan**: Jumlah pembiayaan yang diajukan
   - **Jangka Waktu**: Lama angsuran (bulan)
   - **Tujuan Penggunaan**: Deskripsi penggunaan dana
   - **Agunan**: Informasi jaminan (jika ada)
   - **Upload Dokumen**: KTP, slip gaji, dll

4. Klik "Submit Pengajuan"
5. Tunggu proses verifikasi dari admin

### Status Pengajuan Pembiayaan
- **Diajukan**: Pengajuan baru dibuat, menunggu verifikasi
- **Review**: Sedang diverifikasi oleh admin
- **Disetujui**: Pengajuan disetujui, menunggu pencairan
- **Cair**: Dana sudah dicairkan
- **Ditolak**: Pengajuan ditolak
- **Lunas**: Pembiayaan sudah dilunasi

### Melihat Detail Pembiayaan
1. Klik menu "Pembiayaan"
2. Daftar pembiayaan akan muncul
3. Klik detail untuk melihat:
   - Informasi pokok pembiayaan
   - Jadwal angsuran
   - Riwayat pembayaran
   - Sisa pokok dan margin

### Membayar Angsuran
Angsuran biasanya dipotong otomatis dari:
- Gaji (untuk karyawan)
- Simpanan (untuk anggota biasa)

### Download Bukti Pembayaran
1. Pada detail pembiayaan
2. Klik riwayat pembayaran
3. Klik ikon üñ®Ô∏è untuk print bukti

## Laporan Keuangan

### Jenis Laporan untuk Anggota

#### Laporan Simpanan Pribadi
- Total simpanan per jenis
- Riwayat transaksi
- Saldo terkini

#### Laporan Pembiayaan Pribadi
- Daftar pembiayaan aktif
- Jadwal angsuran
- Riwayat pembayaran
- Sisa tunggakan

### Cara Melihat Laporan
1. Klik menu "Laporan"
2. Pilih jenis laporan
3. Filter berdasarkan periode (tanggal)
4. Klik "Generate Laporan"

### Export Laporan
1. Setelah laporan muncul
2. Klik tombol "Export Excel"
3. File akan terdownload dalam format .xlsx

### Print Laporan
1. Klik tombol "Cetak"
2. Laporan akan terbuka di new tab
3. Gunakan browser print (Ctrl+P)

## Tips dan Panduan Tambahan

### Security Best Practices
- **Password**: Gunakan password yang kuat dan unik
- **Logout**: Selalu logout setelah selesai menggunakan
- **Public Device**: Jangan gunakan di komputer publik
- **Phishing**: Waspadai email penipuan

### Mobile Usage
- Aplikasi responsive untuk mobile
- Gunakan browser Chrome/Safari terbaik
- Portrait mode lebih nyaman untuk navigasi
- Zoom in/out untuk detail transaksi

### Troubleshooting Umum

#### Tidak Bisa Login
- Check koneksi internet
- Verifikasi email dan password
- Clear browser cache
- Reset password jika lupa

#### Data Tidak Muncul
- Refresh halaman (F5)
- Check filter periode
- Pastikan ada transaksi pada periode tersebut
- Contact admin jika masalah berlanjut

#### Download Error
- Check koneksi internet
- Pastikan browser support download
- Coba browser lain
- Check folder download

## FAQ

### Q: Bagaimana cara mengetahui status pengajuan pembiayaan?
A: Login ke akun, menu "Pembiayaan", lihat status pada daftar pembiayaan.

### Q: Bisakah saya menarik simpanan sukarela kapan saja?
A: Ya, simpanan sukarela bisa ditarik dengan mengisi form penarikan dan menunggu persetujuan.

### Q: Berapa lama proses verifikasi pengajuan pembiayaan?
A: Proses verifikasi biasanya 2-3 hari kerja.

### Q: Bagaimana jika saya lupa password?
A: Klik "Lupa Password?" di halaman login, ikuti instruksi reset via email.

### Q: Apakah ada biaya admin untuk transaksi?
A: Tidak, tidak ada biaya admin untuk transaksi simpanan dan pembiayaan.
